Sobre o projeto

O projeto foi desenvolvido no âmbito da disciplina de Tecnologias Web do Curso Técnico Superior Profissional de Tecnologias Web e Dispositivos Móveis. O objetivo do projeto é por em prática e desenvolver os conhecimentos obtidos durante as aulas na criação de um website de tema livre que utilize HTML, CSS ( ou BootStrap ), JavaScript e mesmo uma API ( ou ficheiro JSON ).

Sobre o ficheiro JSON

Foi pedido para que se utilizasse uma API ou ficheiro JSON no website, assim sendo obtou-se pela utilização de um ficheiro JSON. Graças ao ficheiro JSON e à Fecth API foi possível armazenar as informações de hospitais, farmácias e centros de saúde e em seguida recolher-las para as apresentar aos utilizadores.
